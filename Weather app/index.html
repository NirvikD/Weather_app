About Weather App

This is a simple weather application that allows users to check the current weather conditions for a specific city. It provides information such as temperature, wind speed, cloud cover, sunrise, and sunset times.

This application is built using HTML, CSS, and JavaScript. It utilizes the RapidAPI to fetch weather data based on user input.

Feel free to explore and use the application to stay updated with the latest weather information!

User Guide

Welcome to the Weather App user guide! Below are some instructions on how to use the application:

  1. On the home page, you'll see a search bar where you can type the name of the city you want to check the weather for.
  2. After typing the city name, click on the "Search" button.
  3. The application will display the current weather conditions for the specified city, including temperature, wind speed, cloud cover, sunrise, and sunset times.
  4. You can also navigate to the "About" section to learn more about the Weather App.

That's it! Enjoy using the Weather App to stay updated with the latest weather information.
